# HEKTO

- **Database**: [Firebase](https://firebase.google.com/)
- **Payement**: [Stripe](https://stripe.com/)
- **Deployment**: [Vercel](https://vercel.com)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
